## Descripción del programa

Este programa implementa una versión iterativa del algoritmo **MergeSort** utilizando una estructura de árbol binario representada por la clase `MergeNode`.  
Cada nodo combina dos sublistas ordenadas y permite acceder al elemento mínimo disponible mediante los métodos `peek()` y `pop()`.  
La clase `MergeSortIterator` encapsula el proceso de ordenamiento, proporcionando una interfaz iterable que retorna los elementos en orden ascendente.  

## Instrucciones de ejecución

1. Asegúrese de tener instalado el intérprete de Python 3 en su sistema.  
```bash
sudo apt install python3
```
2. Ejecute el programa con el intérprete de Python, por ejemplo:  
```bash
python3 Iterators.py
```
