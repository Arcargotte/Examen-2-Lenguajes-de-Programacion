## Descripción del programa

Este programa implementa el algoritmo de ordenamiento **MergeSort**.  
El proceso consiste en dividir una lista en dos mitades mediante la función `Split`, ordenar cada mitad recursivamente con `MergeSort`,  
y finalmente combinar ambas listas ordenadas con la función `Merge`. El resultado es una lista ordenada en orden ascendente.  
La función `PrintList` permite mostrar los elementos del arreglo resultante en consola.  

## Instrucciones de ejecución

1. Instale el intérprete de Lua desde los repositorios oficiales de Ubuntu.  
```bash
sudo apt install lua5.4
```
2. Ejecute el programa con el intérprete de Lua, por ejemplo:  
```bash
lua Problem2.lua
```
